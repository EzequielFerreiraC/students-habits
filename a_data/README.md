# a_data

Dados brutos utilizados pelo projeto.

- `student_habits_performance.csv` — uma linha por estudante com atributos
  demográficos, métricas diárias de hábitos (estudo, sono, redes sociais,
  exercício, alimentação) e a nota final `exam_score`.

Não commitar cópias modificadas do dataset; use o módulo loader para
gerar versões derivadas em memória.
