# Dataset

O dataset `student_habits_performance.csv` contém uma linha por estudante
com as seguintes colunas:

| Coluna                         | Tipo     | Descrição |
|--------------------------------|----------|-----------|
| `student_id`                   | string   | Identificador único |
| `age`                          | int      | Idade em anos |
| `gender`                       | string   | Gênero |
| `study_hours_per_day`          | float    | Horas diárias de estudo |
| `social_media_hours`           | float    | Horas diárias em redes sociais |
| `netflix_hours`                | float    | Horas diárias em streaming |
| `part_time_job`                | string   | Sim/Não |
| `attendance_percentage`        | float    | Frequência em aula |
| `sleep_hours`                  | float    | Média de sono diário |
| `diet_quality`                 | string   | Poor / Fair / Good |
| `exercise_frequency`           | int      | Sessões por semana |
| `parental_education_level`     | string   | Maior grau dos pais |
| `internet_quality`             | string   | Poor / Average / Good |
| `mental_health_rating`         | int      | Auto-relatado 1-10 |
| `extracurricular_participation`| string   | Sim/Não |
| `exam_score`                   | float    | Nota final (0-100) |
